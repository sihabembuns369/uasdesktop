Apa itu Web Design?

Responsive Web adalah sebuah metode sistem web yang bertujuan untuk mengoptimalkan web jika web tersebut dibuka dalam berbagai ukuran layar. Dengan metode ini, website akan menyesuaikan atau beradaptasi jika dibuka dari perangkat mobile/ponsel berukuran kecil, tablet, ataupun komputer dengan monitor yang besar
